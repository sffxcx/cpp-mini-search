# cpp-mini-search
Высокопроизводительный движок полнотекстового поиска на C++17. Реализует инвертированный индекс, токенизацию и ранжирование результатов по алгоритму TF-IDF. Встроенный HTTP-сервер для запросов, хранение метаданных в SQLite. Демонстрация работы со структурами данных, памятью и сетями без тяжелых зависимостей.
